# Password Generator

A simple and customizable password generator built with React and Tailwind CSS. Easily generate secure passwords with options for uppercase, lowercase, numbers, and special characters.

## Features

- Select password length (4–50 characters)
- Toggle inclusion of uppercase, lowercase, numbers, and special characters
- Show/hide generated password
- Copy password to clipboard
- Reset form to default state
- Responsive and accessible UI
- Toast notifications for actions

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/iLiaiLo/password-generator.git
   cd password-generator
   ```
